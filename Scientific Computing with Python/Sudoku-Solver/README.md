# Sudoku Solver Training

This project is a Sudoku Solver built as part of FreeCodeCamp Python training.

## Description
The solver uses object-oriented programming concepts such as classes and methods to solve a Sudoku puzzle in-place.

## How to Use
1. Open the `Learn Classes and Objects by Building a Sudoku Solver.py` file.
2. Run the Python script.
3. The console will display the unsolved and solved Sudoku puzzle.

## Features
- Finds empty cells in the Sudoku board
- Checks if a number is valid in a row, column, and 3x3 square
- Solves the puzzle recursively using backtracking
