# Build a Book Inventory App

## Project Overview
This certification project is a Book Inventory web application built using semantic HTML tables and advanced CSS styling techniques. The application displays a collection of books with their title, author, category, reading status, and rating system.

The goal of this project is to practice structured data presentation, conditional styling using attribute selectors, and dynamic visual feedback through CSS techniques.

## Features
- Semantic HTML table structure
- Categorized reading status (Read, To Read, In Progress)
- Dynamic row styling based on book status
- Visual rating system using CSS-based circular indicators
- Gradient-based UI design
- Clean and readable data layout

## Technologies Used
- HTML5
- CSS3
- CSS Attribute Selectors
- CSS Gradients
- Pseudo-classes (nth-child)
- Structured Table Layout

## How It Works
- Each row is styled dynamically using class-based selectors.
- Status badges change appearance based on reading progress.
- Rating indicators are built using nested span elements.
- The number of colored circles represents the rating level.
