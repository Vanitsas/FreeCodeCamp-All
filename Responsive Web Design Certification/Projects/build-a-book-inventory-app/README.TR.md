# Kitap Envanteri Uygulaması Oluşturun

## Proje Hakkında
Bu sertifika projesi, semantik HTML tablo yapısı ve gelişmiş CSS teknikleri kullanılarak oluşturulmuş bir Kitap Envanteri uygulamasıdır. Uygulama; kitap başlığı, yazar, kategori, okuma durumu ve puanlama sistemini görsel olarak sunmaktadır.

Bu çalışmanın amacı, yapılandırılmış veri sunumu, attribute selector kullanımı ve CSS ile dinamik görsel geri bildirim oluşturma becerilerini geliştirmektir.

## Özellikler
- Semantik HTML tablo yapısı
- Okuma durumuna göre kategorize edilmiş satırlar (Read, To Read, In Progress)
- Duruma göre dinamik satır arka plan tasarımı
- CSS ile oluşturulmuş görsel puanlama sistemi
- Gradient tabanlı modern arayüz
- Temiz ve okunabilir veri düzeni

## Kullanılan Teknolojiler
- HTML5
- CSS3
- CSS Attribute Selectors
- CSS Gradient
- nth-child pseudo-class
- Yapılandırılmış tablo düzeni

## Nasıl Çalışır
- Her satır, class bazlı selector'lar ile dinamik olarak stillendirilir.
- Okuma durumuna göre durum etiketleri farklı görünümler alır.
- Puanlama sistemi iç içe span elementleri kullanılarak oluşturulmuştur.
- Renkli daire sayısı puan seviyesini temsil eder.
