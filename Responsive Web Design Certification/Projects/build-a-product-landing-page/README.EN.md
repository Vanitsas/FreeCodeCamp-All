# Build a Product Landing Page

This project is part of the **freeCodeCamp Responsive Web Design Certification**. The goal of this project is to create a functional product landing page that meets all the required user stories defined by freeCodeCamp.

The page demonstrates the use of semantic HTML, responsive design techniques, navigation links, embedded media, and a working form submission.

## Project Location in Repository
Responsive Web Design Certification/Projects/build-a-product-landing-page

## Features

- Fixed navigation header
- Navigation links that scroll to sections on the page
- Product feature section
- Embedded video explaining how the product works
- Pricing section
- Email subscription form
- Responsive layout using CSS
- Mobile-friendly navigation using media queries

## Technologies Used

- HTML5
- CSS3
- Responsive Design
- Flexbox
- Media Queries

## freeCodeCamp Requirements Implemented

This project satisfies the following freeCodeCamp user stories:

- The page contains a `header` element with the id `header`
- A logo image with the id `header-img`
- A navigation bar with the id `nav-bar`
- Navigation links with the class `nav-link`
- Navigation links scroll to corresponding sections
- A product video embedded using an `iframe` with the id `video`
- A form element with the id `form`
- An email input field with the id `email`
- A submit button with the id `submit`
- The form submits to the required freeCodeCamp endpoint
- The header remains fixed at the top of the page
- The layout is responsive and adapts to smaller screens
