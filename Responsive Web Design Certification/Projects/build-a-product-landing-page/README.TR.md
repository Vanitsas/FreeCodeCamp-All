# Ürün Tanıtım Sayfası Oluşturun

Bu proje **freeCodeCamp Responsive Web Design Certification** kapsamında yapılmıştır. Projenin amacı freeCodeCamp tarafından belirlenen tüm kullanıcı hikâyelerini karşılayan işlevsel bir ürün tanıtım sayfası oluşturmaktır.

Sayfa; semantik HTML yapısı, responsive tasarım teknikleri, navigasyon bağlantıları, gömülü video ve çalışan bir form yapısını göstermektedir.

## Projenin Repo İçindeki Yeri
Responsive Web Design Certification/Projects/build-a-product-landing-page

## Özellikler

- Sabit (fixed) navigasyon başlığı
- Sayfa içindeki bölümlere giden navigasyon linkleri
- Ürün özellikleri bölümü
- Ürünün nasıl çalıştığını gösteren gömülü video
- Fiyatlandırma bölümü
- E-posta abonelik formu
- CSS ile responsive tasarım
- Media query kullanarak mobil uyumlu yapı

## Kullanılan Teknolojiler

- HTML5
- CSS3
- Responsive Design
- Flexbox
- Media Queries

## Uygulanan freeCodeCamp Gereksinimleri

Bu proje aşağıdaki freeCodeCamp kullanıcı hikâyelerini karşılamaktadır:

- `header` id'sine sahip bir header elementi bulunur
- `header-img` id'sine sahip bir logo resmi vardır
- `nav-bar` id'sine sahip bir navigasyon menüsü vardır
- `nav-link` class'ına sahip navigasyon bağlantıları bulunur
- Navigasyon bağlantıları sayfa içindeki bölümlere gider
- `video` id'sine sahip iframe ile gömülü bir video vardır
- `form` id'sine sahip bir form bulunur
- `email` id'sine sahip e-posta giriş alanı vardır
- `submit` id'sine sahip bir gönderme butonu bulunur
- Form freeCodeCamp’in belirlediği endpoint'e gönderilir
- Header sayfanın üstünde sabit kalır
- Sayfa responsive tasarıma sahiptir

## Bu Projede Pratik Yapılan Konular

Bu proje ile birlikte şu konularda pratik yapılmıştır:

- Gerçekçi bir landing page yapısı kurma
- Sabit navigasyon header oluşturma
- Harici medya gömme (YouTube iframe)
- Form oluşturma ve doğrulama
- Responsive tasarım uygulama
- Düzenli ve temiz HTML / CSS yazma

## Projenin Amacı

Bu proje, **gerçek bir ürün tanıtım sayfası tasarımını** oluşturmayı ve responsive web tasarım prensiplerini pekiştirmeyi amaçlamaktadır.
