# Teknik Dokümantasyon Sayfası Oluşturun

Bu proje, FreeCodeCamp Responsive Web Design Certification kapsamında hazırlanmıştır.

## 📌 Proje Özeti

Bu çalışmada HTML ve CSS kullanarak yapılandırılmış bir teknik dokümantasyon sayfası oluşturulmuştur.  
Amaç:

- Sabit yan menü oluşturmak
- Ana içerik bölümlerini düzenlemek
- Semantik HTML kullanmak
- Kod blokları ve listeleri stilize etmek
- Responsive tasarım ile mobil uyum sağlamak

Sayfa, siber güvenlik konularını kapsayan ve kolay gezilebilir bağlantılar ile örnek kod blokları içeren bir yapı sunar.

## 🧱 Kullanılan Teknolojiler

- HTML5
- CSS3
- Flexbox
- Media Query
- Fixed Position
- Semantik HTML

## 🎯 Pekiştirilen Konular

### Layout
- Sabit yan menü (#navbar)
- Ana içerik (#main-doc) ve çoklu bölümler
- Navigasyon için bağlantılar

### Tipografi ve Stil
- Başlıklar `<header>` ile hiyerarşik yapı
- Açıklamalar için paragraflar
- Kod blokları `<code>` ile stilize edildi
- Liste öğeleri ile maddeleme

### Responsive Tasarım
- 768px altındaki ekranlar → dikey layout
- Navbar relative olur, içerik mobil uyumlu hale gelir
- Tüm cihazlarda okunabilirlik sağlanır

## 🚀 Kazanımlar

- Teknik dokümantasyon sayfalarını etkili şekilde yapılandırmak
- Sabit yan menü ile kaydırmalı içerik oluşturmak
- Semantik HTML öğelerini stilize etmek
- Responsive teknikler ile çok cihazlı destek sağlamak
