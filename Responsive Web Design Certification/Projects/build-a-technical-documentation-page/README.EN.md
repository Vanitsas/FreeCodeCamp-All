# Build a Technical Documentation Page

This project is part of the FreeCodeCamp Responsive Web Design Certification.

## 📌 Project Overview

The goal of this lab is to create a structured technical documentation page using HTML and CSS.  
It focuses on:

- Building a fixed side navigation bar
- Structuring main content sections
- Using semantic HTML tags
- Styling code blocks and lists
- Responsive layout with media queries

The page covers core cyber security topics with easily navigable links and clear code examples.

## 🧱 Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- Fixed Positioning
- Semantic HTML

## 🎯 Key Concepts Practiced

### Layout
- Side navigation (#navbar) with fixed positioning
- Main content (#main-doc) with multiple sections
- Smooth scrolling links for navigation

### Typography & Styling
- Header hierarchy using `<header>`
- Paragraphs for explanation
- Code blocks styled with `<code>`
- Lists for bullet points

### Responsive Design
- On screens under 768px: vertical layout
- Navbar becomes relative and content adapts
- Maintains readability on mobile and tablet devices

## 🚀 What I Learned

- How to structure technical documentation pages effectively
- Implementing a fixed sidebar with scrolling content
- Styling semantic HTML elements for clarity
- Using responsive techniques for multi-device support
