# Build a Survey Form

This project is part of freeCodeCamp's Responsive Web Design curriculum.

The purpose of this project is to practice building a functional HTML form that collects user information. The form includes fields for full name, email, age, favorite topics, preferred platform, tools used, and additional comments.

## What You Learn
- Creating HTML forms
- Using different input types: text, email, number
- Dropdown select menus
- Radio buttons and checkboxes
- Textareas for longer input
- Grouping related fields using fieldset and legend
- Form submission basics

## Technologies Used
- HTML5
