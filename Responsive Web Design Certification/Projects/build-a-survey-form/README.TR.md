# Anket Formu Oluşturun

Bu proje, freeCodeCamp'ın Responsive Web Design müfredatının bir parçasıdır.

Bu çalışmanın amacı, kullanıcı bilgilerini toplayan işlevsel bir HTML formu oluşturma pratiği yapmaktır. Form, tam isim, e-posta, yaş, favori konular, tercih edilen platform, kullanılan araçlar ve ek yorumlar gibi alanları içerir.

## Öğrenilen Konular
- HTML formları oluşturma
- Farklı input tiplerini kullanma: text, email, number
- Dropdown select menüleri
- Radio button ve checkbox kullanımı
- Uzun metinler için textarea kullanımı
- İlgili alanları gruplamak için fieldset ve legend kullanımı
- Form gönderme temelleri

## Kullanılan Teknolojiler
- HTML5
