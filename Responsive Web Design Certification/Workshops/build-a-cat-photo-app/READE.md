# Cat Photo App

A simple HTML project built as part of a FreeCodeCamp challenge.

## Features

- Uses semantic HTML elements: `<section>`, `<figure>`, `<figcaption>`, `<em>`, `<strong>`.
- Nested elements and proper indentation for readability.
- Lists (`<ul>` and `<ol>`) for structured content.
- Links (`<a>`) for text and images.
- Footer linking to [freeCodeCamp](https://www.freecodecamp.org).

## How to View

1. Open `cat-photo-app.html` in your browser.
2. Enjoy the Cat Photo App!
