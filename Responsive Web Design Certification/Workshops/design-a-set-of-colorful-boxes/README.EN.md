# Design a Set of Colorful Boxes

## Project Location in Repository
Responsive Web Design Certification/Workshops/design-a-set-of-colorful-boxes

## Description
This workshop demonstrates how to create a responsive layout using CSS Flexbox. The page contains several colorful boxes arranged inside a flexible container that adapts to different screen sizes.

Each box represents a color and uses Flexbox properties such as `flex-grow`, `flex-shrink`, `flex-wrap`, and `order` to control layout behavior.

## Features
- Flexbox layout container
- Multiple boxes with different colors
- Use of `order` to rearrange elements
- Demonstration of `flex-grow` and `flex-shrink`
- Responsive wrapping using `flex-wrap`
- Centered page layout

## Technologies Used
- HTML5
- CSS3
- Flexbox Layout

## What I Learned
- How to structure layouts with Flexbox
- How `order` changes the visual position of elements
- How `flex-grow` and `flex-shrink` affect element size
- How to create responsive layouts using `flex-wrap`
