# Renkli Kutular Tasarlama

## Projenin Repo İçindeki Yeri
Responsive Web Design Certification/Workshops/design-a-set-of-colorful-boxes

## Açıklama
Bu workshop, CSS Flexbox kullanarak esnek ve düzenli bir sayfa yapısı oluşturmayı gösterir. Sayfa içinde farklı renklere sahip kutular bir Flexbox konteyneri içinde yer alır ve ekran boyutuna göre düzenlenebilir.

Her kutu bir rengi temsil eder ve yerleşimi kontrol etmek için `flex-grow`, `flex-shrink`, `flex-wrap` ve `order` gibi Flexbox özellikleri kullanılır.

## Özellikler
- Flexbox tabanlı layout
- Farklı renklerde kutular
- `order` özelliği ile elemanların sırasını değiştirme
- `flex-grow` ve `flex-shrink` kullanımı
- `flex-wrap` ile responsive düzen
- Ortalanmış sayfa yapısı

## Öğrenilenler
- Flexbox ile sayfa düzeni oluşturma
- `order` özelliği ile görsel sıralamayı değiştirme
- `flex-grow` ve `flex-shrink` ile eleman boyutunu kontrol etme
- `flex-wrap` ile responsive layout oluşturma
