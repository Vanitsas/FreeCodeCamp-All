# Heart Icon

This project is a simple workshop from FreeCodeCamp where we build a **heart icon** using SVG.  

## Features

- Small 24x24 heart icon
- Red color
- Pure SVG, no external libraries

## Usage

You can open the `index.html` file in any browser to see the heart icon.  
You can also use the `svg` code directly in your projects as an icon.

```html
<svg width="24" height="24" viewBox="0 0 24 24" fill="red">
  <path d="M12 21s-6-4.35-9.33-8.22C-.5 7.39 3.24 1 8.4 4.28 10.08 5.32 12 7.5 12 7.5s1.92-2.18 3.6-3.22C20.76 1 24.5 7.39 21.33 12.78 18 16.65 12 21 12 21z"/>
</svg>
