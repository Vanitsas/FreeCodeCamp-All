# Build a Flexbox Photo Gallery

This workshop demonstrates how to create a responsive photo gallery using CSS Flexbox.  

## Features

- Responsive layout with flex-wrap and justify-content  
- Centered images with equal spacing  
- Rounded corners on images  
- Maximum width for gallery container  
- Header with custom styling  

## Usage

1. Open `index.html` in your browser  
2. View the gallery layout with images arranged in a flexible grid  
3. Customize styles in `styles.css`
