# Flexbox Fotoğraf Galerisi Oluşturma

Bu workshop, CSS Flexbox kullanarak duyarlı bir fotoğraf galerisi oluşturmayı gösterir.  

## Özellikler

- flex-wrap ve justify-content ile duyarlı düzen  
- Eşit boşluklarla ortalanmış resimler  
- Resimlerde yuvarlatılmış köşeler  
- Galeri konteyneri için maksimum genişlik  
- Özel stillere sahip başlık  

## Kullanım

1. `index.html` dosyasını tarayıcınızda açın  
2. Resimlerin esnek bir ızgara içinde düzenlendiği galeriyi görüntüleyin  
3. Stil özelleştirmeleri için `styles.css` dosyasını kullanın
