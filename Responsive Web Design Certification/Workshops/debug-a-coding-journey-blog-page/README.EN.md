# Debug a Coding Journey Blog Page

This project is part of freeCodeCamp's Responsive Web Design curriculum.

The goal of this workshop is to practice building a semantic HTML blog page. The page includes multiple articles, headings, a navigation menu, and a footer with contact information.

## What You Learn
- Structuring a blog page with semantic HTML
- Using `<header>`, `<main>`, `<article>`, `<nav>`, `<footer>`
- Organizing content with headings and sections
- Creating accessible links and email contacts
- Using IDs for navigation anchors

## Technologies Used
- HTML5
