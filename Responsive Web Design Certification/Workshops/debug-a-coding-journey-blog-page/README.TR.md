# Kodlama Yolculuğunda Hata Ayıklama Blog Sayfası

Bu proje, freeCodeCamp'ın Responsive Web Design müfredatının bir parçasıdır.

Bu atölyenin amacı, semantik HTML kullanarak bir blog sayfası oluşturma pratiği yapmaktır. Sayfa birden fazla makale, başlıklar, navigasyon menüsü ve iletişim bilgilerini içeren bir footer içerir.

## Öğrenilen Konular
- Semantik HTML ile blog sayfası yapılandırma
- `<header>`, `<main>`, `<article>`, `<nav>`, `<footer>` etiketlerini kullanma
- İçeriği başlıklar ve bölümlerle organize etme
- Erişilebilir linkler ve e-posta kontakları oluşturma
- Navigasyon için ID kullanımı

## Kullanılan Teknolojiler
- HTML5
