# Bir Şehir Silüeti Oluşturun

Bu workshop, FreeCodeCamp Responsive Web Design Certification kapsamında yapılmıştır.

## Proje Özeti
Bu projede yalnızca HTML ve CSS kullanılarak stilize edilmiş bir şehir silüeti oluşturulmuştur. Şehir silüeti arka plan ve ön plan binalarından oluşur. CSS gradientleri, değişkenler ve layout teknikleri kullanılarak binalar, pencereler ve gökyüzü görsel olarak oluşturulmuştur.

## Kullanılan Teknolojiler
- HTML5
- CSS3
- CSS Variables
- Flexbox
- Linear Gradient
- Radial Gradient
- Responsive Design

## Öğrenilen Konular
- `:root` içinde tanımlanan CSS değişkenlerini kullanmak
- Flexbox ile layout oluşturmak
- Birden fazla HTML elemanı kullanarak görsel yapı oluşturmak
- `linear-gradient` ve `repeating-linear-gradient` kullanmak
- Katmanlı görsel tasarım oluşturmak
- Media query ile responsive tasarım yapmak

## Responsive Davranış
Ekran genişliği 1000px altına düştüğünde CSS değişkenleri değiştirilerek tasarım daha koyu renklere dönüşür ve gece şehir görünümü oluşur.

## Kazanımlar
- HTML ve CSS ile görsel layout oluşturma
- CSS değişkenleri ile tekrar kullanılabilir stil yazma
- Gradient kullanarak mimari detaylar üretme
- Ön plan ve arka plan katmanları oluşturma
- Media query ile responsive değişiklikler yapma
