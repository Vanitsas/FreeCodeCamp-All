# Build a City Skyline

This workshop is part of the FreeCodeCamp Responsive Web Design Certification.

## Project Overview
In this project, a stylized city skyline is created using only HTML and CSS. The skyline consists of background buildings and foreground buildings constructed from multiple nested elements. CSS gradients, variables, and layout techniques are used to visually simulate buildings, windows, and the sky.

## Technologies Used
- HTML5
- CSS3
- CSS Variables
- Flexbox
- Linear Gradients
- Radial Gradients
- Responsive Design

## Key Concepts
- Using CSS custom properties defined in `:root`
- Creating layouts with Flexbox
- Building visual components using nested elements
- Using `linear-gradient` and `repeating-linear-gradient` for patterns
- Creating layered visual compositions
- Applying responsive design with media queries

## Responsive Behavior
When the screen width becomes smaller than 1000px, the design switches to a darker theme by changing CSS variables. This simulates a nighttime skyline.

## What I Learned
- Structuring visual layouts using HTML and CSS
- Using CSS variables for reusable styling
- Combining gradients to simulate architectural details
- Creating layered foreground and background elements
- Implementing responsive visual changes with media queries
