# Teknoloji Konferansı Program Tablosu Oluşturun

Bu proje, freeCodeCamp'ın Responsive Web Design müfredatının bir parçasıdır.

Bu atölyenin amacı, bir teknoloji konferansı programını göstermek için semantik bir HTML tablo oluşturma pratiği yapmaktır. Sayfa, başlıklar, satırlar, colspan ve scope öznitelikleri içeren bir tablo içerir; bu sayede erişilebilirlik artırılır.

## Öğrenilen Konular
- HTML `<table>` etiketleri ile veri yapılandırma
- `<thead>`, `<tbody>`, `<tfoot>` kullanımı
- Erişilebilirlik için `colspan` ve `scope` özniteliklerini uygulama
- Tablo içeriğini açıklamak için `<caption>` kullanımı

## Kullanılan Teknolojiler
- HTML5
