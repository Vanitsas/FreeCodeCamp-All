# Build a Tech Conference Schedule Table

This project is part of freeCodeCamp's Responsive Web Design curriculum.

The goal of this workshop is to practice building a semantic HTML table to display a tech conference schedule. The page includes a table with headers, rows, colspan, and scope attributes to improve accessibility.

## What You Learn
- Structuring data using HTML `<table>` elements
- Using `<thead>`, `<tbody>`, `<tfoot>`
- Applying `colspan` and `scope` attributes for accessibility
- Adding a `<caption>` to describe table content

## Technologies Used
- HTML5
