# Build Quincy's Job Tips Page

Bu proje, freeCodeCamp HTML workshop çalışmasının bir parçası olarak hazırlanmıştır.

Bu çalışmanın amacı, temel HTML yapısını öğrenmek ve semantik öğeler kullanarak bir sayfa oluşturmayı pratize etmektir. Özellikle `blockquote`, `q`, `main` ve `section` öğeleri üzerinde durulmuştur.

## Öğrenilen Konular
- HTML5 sayfa yapısı
- `<blockquote>` ve `<q>` kullanımı
- Semantik ve temiz HTML yazımı
- Başlıklar ve içerik bölümlendirme (`h1`, `h2`, `section`, `main`)

## Kullanılan Teknolojiler
- HTML5
