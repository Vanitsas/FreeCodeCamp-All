# Cafe Menu Tasarımı

Bu proje, freeCodeCamp'in Responsive Web Design müfredatının bir parçasıdır.

Bu workshop’un amacı, bir cafe menü sayfası oluşturma pratiği yapmaktır. Sayfa, kahve ve tatlı ürünlerini fiyatlarıyla, resimler, başlıklar ve iletişim bilgileri içeren bir footer ile gösterir.

## Öğrendikleriniz
- Semantic HTML ile menü sayfası yapısı oluşturmak
- `<main>`, `<section>`, `<article>`, `<footer>` kullanımı
- Başlıklar ve resimlerle içeriği düzenlemek
- CSS ile öğeleri hizalamak (inline-block, block, margin auto)
- Linkler için pseudo-class kullanımı (`:visited`, `:hover`, `:active`)
- Margin ve negatif margin ile boşlukları kontrol etmek
- Arka plan resmi kullanarak sayfayı stilize etmek

## Kullanılan Teknolojiler
- HTML5  
- CSS3

## Özellikler
- Kahve ve tatlı menüleri, ürün adı ve fiyatlarıyla
- Kahve ve tatlılar için dekoratif resimler
- Sayfa için arka plan resmi
- Footer içinde iletişim bilgileri ve harici link
- CSS ile responsive düzen
