# Design a Cafe Menu

This project is part of freeCodeCamp's Responsive Web Design curriculum.

The goal of this workshop is to practice building a cafe menu page. The page includes coffee and dessert items with prices, images, headings, and a footer with contact information.

## What You Learn
- Structuring a menu page with semantic HTML
- Using `<main>`, `<section>`, `<article>`, `<footer>`
- Organizing content with headings and images
- Aligning elements using CSS (inline-block, block, margin auto)
- Applying pseudo-classes for links (`:visited`, `:hover`, `:active`)
- Controlling spacing with margins and negative margins
- Using background images for styling

## Technologies Used
- HTML5  
- CSS3

## Features
- Coffee and dessert menus with flavors and prices
- Decorative images for coffee and desserts
- Background image for the page
- Footer with contact information and external link
- Responsive layout using CSS
