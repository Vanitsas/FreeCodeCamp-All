# Build a Balance Sheet

## Project Overview
This project is a financial layout workshop where a company balance sheet is built using semantic HTML tables and modern CSS styling techniques. The page presents Assets, Liabilities, and Net Worth data across multiple years.

The goal of this project is to practice structured table layouts, accessibility techniques, responsive design, and visual hierarchy using CSS.

## Features
- Semantic HTML table structure
- Multiple financial sections (Assets, Liabilities, Net Worth)
- Sticky year header
- Accessible screen-reader-only elements
- Styled totals with hover effects
- Clean financial data alignment
- Responsive layout structure

## Technologies Used
- HTML5
- CSS3
- CSS Flexbox
- CSS Sticky Positioning
- Accessibility techniques (sr-only pattern)
- Linear gradients
