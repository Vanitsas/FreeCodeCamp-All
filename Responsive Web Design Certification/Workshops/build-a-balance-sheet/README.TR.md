# Bilanço Oluşturun

## Proje Hakkında
Bu proje, bir şirketin bilançosunu (Balance Sheet) semantik HTML tablo yapısı ve modern CSS teknikleri kullanarak oluşturmayı amaçlayan bir çalışmadır. Sayfa; Varlıklar (Assets), Yükümlülükler (Liabilities) ve Net Değer (Net Worth) verilerini farklı yıllar için göstermektedir.

Bu çalışmanın amacı tablo yapıları, erişilebilirlik (accessibility), responsive tasarım ve görsel hiyerarşi konularında pratik yapmaktır.

## Özellikler
- Semantik HTML tablo yapısı
- Birden fazla finansal bölüm (Assets, Liabilities, Net Worth)
- Sticky (sabit) yıl başlığı
- Screen reader için erişilebilir gizli metin yapısı
- Hover efekti olan toplam satırları
- Düzenli ve profesyonel veri hizalaması
- Responsive yapı

## Kullanılan Teknolojiler
- HTML5
- CSS3
- CSS Flexbox
- CSS Sticky Positioning
- Accessibility teknikleri (sr-only yöntemi)
- Linear Gradient
