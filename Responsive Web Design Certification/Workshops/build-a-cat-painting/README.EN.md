# Build a Cat Painting

## Project Overview
This project is a CSS art workshop where a cat face is created entirely using HTML and CSS. The illustration is built using div elements, positioning, borders, gradients, and transforms without using any images.

The purpose of this project is to improve CSS positioning skills, shape creation techniques, and creative layout control.

## Features
- Pure CSS drawing (no images used)
- Symmetrical ear and eye structure
- Gradient background for depth
- Custom nose and mouth built with border techniques
- Whiskers created using positioned div elements
- Centered layout using absolute positioning

## Technologies Used
- HTML5
- CSS3
- CSS Gradients
- CSS Transform (rotate)
- Absolute Positioning
