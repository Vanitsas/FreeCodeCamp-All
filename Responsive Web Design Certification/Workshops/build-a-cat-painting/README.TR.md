# Bir Kedi Resmi Oluşturun

## Proje Hakkında
Bu proje, yalnızca HTML ve CSS kullanılarak oluşturulmuş bir CSS sanat çalışmasıdır. Kedi yüzü tamamen div elementleri, konumlandırma, border teknikleri, gradient ve transform özellikleri kullanılarak tasarlanmıştır. Herhangi bir görsel kullanılmamıştır.

Bu çalışmanın amacı, CSS konumlandırma becerilerini geliştirmek ve şekil oluşturma tekniklerini pekiştirmektir.

## Özellikler
- Saf CSS çizim (görsel kullanılmadı)
- Simetrik kulak ve göz yapısı
- Derinlik için gradient arka plan
- Border teknikleri ile oluşturulmuş burun ve ağız
- Konumlandırılmış div elementleri ile yapılmış bıyıklar
- Absolute positioning ile ortalanmış tasarım

## Kullanılan Teknolojiler
- HTML5
- CSS3
- CSS Gradient
- CSS Transform (rotate)
- Absolute Positioning
