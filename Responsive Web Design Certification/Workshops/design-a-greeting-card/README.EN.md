# Design a Greeting Card

This project was completed as part of a **FreeCodeCamp workshop exercise**.

## Overview
A stylized greeting card webpage with interactive sections.  
The card includes a birthday message, links to send or share, and hover/target effects.

## Features
- Card component with hover effects
- Decorative emojis on the header
- Send and Share links with hover, focus, and active states
- Interactive sections revealed on target
- Clean, responsive layout
- External CSS styling

## Built With
- HTML5
- CSS3
