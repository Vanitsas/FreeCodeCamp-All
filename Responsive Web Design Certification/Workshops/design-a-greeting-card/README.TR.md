# Tebrik Kartı Tasarlayın

Bu proje, **FreeCodeCamp workshop çalışması** kapsamında hazırlanmıştır.

## Genel Bakış
İnteraktif bölümlere sahip stilize bir tebrik kartı web sayfası.  
Kart, doğum günü mesajı içerir ve gönderme / paylaşma linkleri ile hover/target efektleri uygular.

## Özellikler
- Hover efektli kart komponenti
- Başlıkta dekoratif emojiler
- Gönder ve Paylaş linklerinde hover, focus ve active durumları
- Target ile görünür olan interaktif bölümler
- Temiz, responsive tasarım
- Harici CSS kullanımı

## Kullanılan Teknolojiler
- HTML5
- CSS3
