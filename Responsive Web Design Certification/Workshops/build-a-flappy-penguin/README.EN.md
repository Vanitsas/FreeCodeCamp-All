# build-a-flappy-penguin

## Project Overview
This project is a fun interactive web page that displays a colorful flappy penguin character. It demonstrates CSS animations, gradients, and positioning to create a visually engaging scene with mountains, sun, and animated penguin features.

## Project Structure
build-a-flappy-penguin/
├── index.html
├── styles.css

## Features
- **Animated Penguin:** The penguin reacts when clicked and has animated arms.
- **Colorful Background:** Mountains, sun, and ground created using CSS gradients.
- **Detailed Penguin Design:** Includes head, eyes, beak, blush, body, arms, and feet.
- **Interactive:** Penguin scales up on click with smooth transitions.
- **Responsive Design:** Works across different screen sizes.

## How to Use
1. Open `index.html` in your browser.  
2. Edit the HTML and CSS to customize your penguin or background.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Animations, Gradients)
- Responsive Design principles
