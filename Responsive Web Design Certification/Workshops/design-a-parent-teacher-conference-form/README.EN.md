# Design a Parent Teacher Conference Form

This project was created as part of a **FreeCodeCamp workshop**.

## Overview
A responsive and accessible parent-teacher conference form that collects student and parent information, preferred contact method, and additional notes.

## Features
- Structured and semantic HTML form
- Fieldset and legend for accessibility
- Custom styled radio buttons
- Responsive layout
- Modern UI with clean design
- External CSS styling
- Form validation with required fields

## Built With
- HTML5
- CSS3
