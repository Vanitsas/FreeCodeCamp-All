# Animasyonlu Dönme Dolap Yapımı

Bu proje freeCodeCamp Responsive Web Design müfredatının bir parçasıdır. Projenin amacı CSS animasyonlarını ve konumlandırmayı kullanarak dönen bir dönme dolap oluşturmaktır.

Sayfa yapısı HTML ile oluşturulmuş, stil ve animasyonlar CSS ile uygulanmıştır. Dönme dolap CSS keyframe animasyonu ile sürekli dönerken kabinler ters yönde dönerek dik konumlarını korur.

Bu proje, sadece CSS kullanarak animasyonların, dönüşlerin ve konumlandırmanın nasıl birlikte kullanılabileceğini göstermektedir.

## Özellikler

- Dönen dönme dolap animasyonu
- CSS keyframe animasyonları
- Kabinlerin ters yönde dönmesi
- Viewport birimleri ile responsive boyutlandırma
- Absolute positioning ve transform-origin kullanımı

## Kullanılan Teknolojiler

- HTML5
- CSS3
- CSS Animations
- CSS Transform
- Keyframes

## Nasıl Çalışır

Dönme dolap kapsayıcısı keyframe animasyonu ile sürekli döner. Her kabin elemanı ise ters yönde dönen ayrı bir animasyona sahiptir. Bu sayede dönme dolap dönerken kabinler doğal şekilde dik kalır.

Çizgiler CSS transform kullanılarak tekerleğin kollarını oluşturur. Kabinler ise absolute positioning kullanılarak çember etrafına yerleştirilir.
