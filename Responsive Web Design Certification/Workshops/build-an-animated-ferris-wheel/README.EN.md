# Build an Animated Ferris Wheel

This project is part of the freeCodeCamp Responsive Web Design curriculum. The goal of the project is to practice CSS animations and positioning by creating a rotating Ferris wheel.

The layout uses HTML for structure and CSS for styling and animations. The wheel rotates continuously using CSS keyframes, while the cabins counter-rotate to remain upright during the animation.

The project demonstrates how animations, transforms, and positioning can be combined to build visually dynamic UI components using pure CSS.

## Features

- Rotating Ferris wheel animation
- CSS keyframe animations
- Cabin counter-rotation effect
- Responsive sizing using viewport units
- Absolute positioning and transform-origin usage

## Technologies Used

- HTML5
- CSS3
- CSS Animations
- CSS Transform
- Keyframes

## How It Works

The Ferris wheel container rotates using a keyframe animation. Each cabin element has its own animation that rotates in the opposite direction to maintain a natural upright position as the wheel spins.

Lines are positioned using CSS transforms to simulate the wheel spokes, while cabins are positioned around the circle using absolute positioning.
