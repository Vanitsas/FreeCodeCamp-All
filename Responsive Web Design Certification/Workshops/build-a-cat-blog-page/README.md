# Build a Cat Blog Page

This project is part of a freeCodeCamp HTML workshop.

The purpose of this project is to practice semantic HTML and build a simple blog page. The focus is on using elements like `header`, `main`, `footer`, `section`, `article`, and `figure`.

## What You Learn
- HTML5 page structure
- Semantic HTML elements (`header`, `main`, `footer`, `section`, `article`, `figure`)
- Headings and content hierarchy (`h1`, `h2`, `h3`)
- Links and contact information (`a`, `address`)

## Technologies Used
- HTML5
