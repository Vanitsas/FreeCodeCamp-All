# Kedi Blog Sayfası Oluşturma

Bu proje, freeCodeCamp HTML workshop çalışmasının bir parçası olarak hazırlanmıştır.

Bu çalışmanın amacı, semantik HTML kullanarak basit bir blog sayfası oluşturmaktır. Özellikle `header`, `main`, `footer`, `section`, `article` ve `figure` öğeleri üzerinde durulmuştur.

## Öğrenilen Konular
- HTML5 sayfa yapısı
- Semantik HTML öğeleri (`header`, `main`, `footer`, `section`, `article`, `figure`)
- Başlıklar ve içerik hiyerarşisi (`h1`, `h2`, `h3`)
- Linkler ve iletişim bilgileri (`a`, `address`)

## Kullanılan Teknolojiler
- HTML5
