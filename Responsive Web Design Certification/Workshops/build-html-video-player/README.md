# Build an HTML Video Player

This project was created as part of the freeCodeCamp curriculum.

## Description
A simple HTML5 video player built using the `<video>` element.  
The project demonstrates how to work with multiple video formats, MIME types, and common video attributes.

## Features
- HTML5 `<video>` element
- Multiple video sources (mp4, webm, ogg, mov)
- MIME type support
- Video controls
- Loop and muted playback
- Poster thumbnail

## Technologies Used
- HTML5
