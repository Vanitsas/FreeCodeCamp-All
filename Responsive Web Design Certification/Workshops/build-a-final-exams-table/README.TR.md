# Final Sınavları Tablosu Oluşturun

Bu proje, freeCodeCamp HTML atölye çalışmasının bir parçası olarak hazırlanmıştır.

Bu çalışmanın amacı, semantik ve düzenli HTML tabloları oluşturmayı pratik etmektir. Tablo verilerinin anlaşılır ve erişilebilir şekilde sunulması hedeflenmiştir.

## Öğrenilen Konular
- HTML tablo yapısı
- `table`, `caption`, `thead`, `tbody` ve `tfoot` kullanımı
- Tablo satırları ve hücreleri (`tr`, `th`, `td`)
- Özet satırları için `colspan` kullanımı

## Kullanılan Teknolojiler
- HTML5
