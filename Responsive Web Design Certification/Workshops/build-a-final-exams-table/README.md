# Build a Final Exams Table

This project is part of a freeCodeCamp HTML workshop.

The purpose of this project is to practice creating semantic and well-structured HTML tables. The focus is on using table-related elements correctly to display tabular data in a clear and accessible way.

## What You Learn
- HTML table structure
- Using `table`, `caption`, `thead`, `tbody`, and `tfoot`
- Table rows and cells (`tr`, `th`, `td`)
- Creating summary rows with `colspan`

## Technologies Used
- HTML5
