# Build a Bookstore Page

This is a simple HTML project created as part of the FreeCodeCamp curriculum.  
The goal of this project was to build a basic bookstore page using `div`s, `classes`, and `ids` to organize content.

## Features

- Two book cards with titles, descriptions, and "Buy Now" buttons
- A paragraph guiding the user to review selections
- Navigation buttons: "View Cart" and "Checkout"

## Technologies Used

- HTML5

## How to View

1. Clone or download this repository.
2. Open the `index.html` file in your web browser.
