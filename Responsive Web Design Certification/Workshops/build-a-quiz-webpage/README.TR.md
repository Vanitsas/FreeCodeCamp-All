# Bir Sınav Web Sayfası Oluşturun

Bu, freeCodeCamp Accessibility Quiz uygulama projesidir.

## Proje Özeti

Proje, semantik HTML ve erişilebilir form tasarımını gösterir. İçerik:

- Logo, başlık ve gezinme menüsü ile üst bilgi (header)
- Doğru şekilde etiketlenmiş öğrenci bilgileri form alanları
- HTML ve CSS quiz bölümleri (doğru/yanlış ve çoktan seçmeli sorular)
- Smooth scroll ve duyarlı tasarım (responsive)
- İletişim bilgileri ile alt bilgi (footer)

## Nasıl Kullanılır

1. `index.html` dosyasını bir web tarayıcısında açın.
2. Quizi doldurun ve "Send" butonuna tıklayarak gönderin.
3. Form, [freeCodeCamp uygulama projesine](https://freecodecamp.org/practice-project/accessibility-quiz) gönderilir.

## Kullanılan Teknolojiler

- HTML5
- CSS3 (Flexbox, media queries, responsive tasarım)
- Erişilebilirlik özellikleri (aria öznitelikleri, semantik öğeler, ekran okuyucu desteği)
