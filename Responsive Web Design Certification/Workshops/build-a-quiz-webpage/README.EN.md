# Build a Quiz Webpage

This is a freeCodeCamp Accessibility Quiz practice project.

## Project Overview

The project demonstrates semantic HTML and accessible form design. It includes:

- Header with logo, title, and navigation
- Student information section with properly labeled form fields
- HTML and CSS quiz sections with true/false and multiple-choice questions
- Smooth scrolling and responsive layout
- Footer with contact information

## How to Use

1. Open `index.html` in a web browser.
2. Fill out the quiz and submit using the "Send" button.
3. The form is submitted to [freeCodeCamp practice project](https://freecodecamp.org/practice-project/accessibility-quiz).

## Technologies Used

- HTML5
- CSS3 (Flexbox, media queries, responsive design)
- Accessibility features (aria attributes, semantic elements, screen reader support)
