# Build an Accessible Audio Controller

This workshop is part of freeCodeCamp's Responsive Web Design curriculum.

In this workshop, you will create an audio controller that is accessible to all users, including those using assistive technologies. You will practice proper labeling, ARIA attributes, and intuitive UI design for audio controls.

## What You Will Learn
- Creating accessible HTML audio controls
- Using ARIA attributes such as aria-labelledby
- Designing user-friendly and inclusive interfaces

## Technologies Used
- HTML5
