# build-an-html-music-player

## Project Overview
This project is a simple HTML music player that demonstrates how to use the HTML `<audio>` element to play audio files directly in the browser. It includes multiple tracks with built-in playback controls such as play, pause, and loop.

## Project Structure
build-an-html-music-player/
└── index.html

## Features
- **HTML Audio Element:** Uses the `<audio>` tag to play music files.
- **Playback Controls:** Built-in browser controls for play, pause, and volume.
- **Loop Functionality:** Tracks automatically repeat using the loop attribute.
- **Multiple Songs:** Displays multiple songs with their artist information.

## How to Use
1. Open `index.html` in your browser.
2. Click the play button on any track to start listening.

## Technologies Used
- HTML5
- HTML Audio Element
