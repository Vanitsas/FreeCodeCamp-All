# Otel Geri Bildirim Formu Oluşturun

Bu proje, freeCodeCamp HTML Lab çalışmasının bir parçası olarak hazırlanmıştır.

Bu çalışmanın amacı, semantik ve erişilebilir formlar oluşturmayı HTML ile pratik etmektir. Özellikle `form`, `fieldset`, `legend`, `label`, `input`, `select`, `textarea` ve `button` öğeleri üzerinde durulmuştur.

## Öğrenilen Konular
- HTML form yapısı
- Semantik ve erişilebilir form öğeleri
- Input tipleri: text, email, number, radio, checkbox
- Select dropdown ve textarea
- Form gönderimi

## Kullanılan Teknolojiler
- HTML5
