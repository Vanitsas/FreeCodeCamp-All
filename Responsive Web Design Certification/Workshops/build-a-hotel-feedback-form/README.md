# Build a Hotel Feedback Form

This project is part of a freeCodeCamp HTML Lab.

The purpose of this project is to practice creating accessible and semantic forms using HTML. The focus is on elements like `form`, `fieldset`, `legend`, `label`, `input`, `select`, `textarea`, and `button`.

## What You Learn
- HTML form structure
- Semantic and accessible form elements
- Input types: text, email, number, radio, checkbox
- Select dropdowns and textarea
- Form submission

## Technologies Used
- HTML5
