# Önemli Web Tarayıcılarının Listesi

Bu proje, freeCodeCamp HTML atölye çalışmasının bir parçası olarak hazırlanmıştır.

Bu çalışmanın amacı, temel HTML belge yapısını öğrenmek ve tanım listeleri (`dl`, `dt`, `dd`) kullanarak web tarayıcıları hakkında bilgileri düzenli bir şekilde göstermektir.

## Öğrenilen Konular
- HTML5 belge yapısı
- Tanım listelerinin kullanımı (`dl`, `dt`, `dd`)
- Semantik ve temiz HTML yazımı

## Kullanılan Teknolojiler
- HTML5
