# Build a Magazine

This project demonstrates a magazine-style layout with a hero section, text content, images, and social media links. The layout is responsive and adjusts to different screen sizes.

## Features

- **Hero Section**: Includes a logo, title, subtitle, and author information.
- **Text Content**: Paragraphs explaining the new project-driven curriculum, with blockquotes for emphasis.
- **Image Section**: Project images with captions and quotes, arranged in a responsive grid.
- **Social Media Links**: Icons linking to Facebook, Twitter, Instagram, LinkedIn, and YouTube.
- **Responsive Design**: Adjusts layout for mobile devices using CSS Grid and media queries.
- **Typography**: Uses Google Fonts (`Anton`, `Baskervville`, `Raleway`) for headings and body text.
- **Styling**: Hero title and quotes highlighted with accent colors, text justified, and first letters styled in the first paragraph.

## Usage

Open `index.html` in a browser to view the magazine layout. The page is fully responsive, and images resize automatically for smaller screens.
