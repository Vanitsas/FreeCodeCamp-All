# Dergi Tasarımı

Bu proje, dergi tarzı bir sayfa düzeni gösterir; başlık bölümü, metin içerikleri, resimler ve sosyal medya bağlantılarını içerir. Düzen responsive olup farklı ekran boyutlarına uyum sağlar.

## Özellikler

- **Başlık Bölümü**: Logo, başlık, alt başlık ve yazar bilgilerini içerir.
- **Metin İçeriği**: Proje tabanlı yeni müfredat açıklamaları, vurgular için blok alıntılar.
- **Resim Bölümü**: Proje resimleri, altyazılar ve alıntılar, responsive grid ile düzenlenmiş.
- **Sosyal Medya Bağlantıları**: Facebook, Twitter, Instagram, LinkedIn ve YouTube ikonları.
- **Responsive Tasarım**: CSS Grid ve media query ile mobil cihazlara uyumlu.
- **Tipografi**: Başlıklar ve metin için Google Fonts (`Anton`, `Baskervville`, `Raleway`) kullanılmıştır.
- **Stil**: Başlık ve alıntılar vurgu renkleriyle öne çıkarılmış, metin justify edilmiş ve ilk paragrafın ilk harfi özel stil verilmiştir.

## Kullanım

`index.html` dosyasını tarayıcıda açarak dergi tasarımını görüntüleyebilirsiniz. Sayfa tamamen responsive olup, resimler ekran boyutuna göre otomatik olarak yeniden boyutlanır.
