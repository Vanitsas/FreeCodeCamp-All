# Build a House Painting

## Project Overview
This project is a CSS art lab where a complete house scene is created using only HTML and CSS. The design includes a roof, chimney with animated smoke, windows, door, bushes, sky gradient, and moving clouds.

The purpose of this project is to improve CSS layout control, positioning skills, animations, and creative visual design techniques.

## Features
- Pure HTML & CSS illustration (no images used)
- Animated chimney smoke
- Moving clouds using CSS keyframes
- Gradient sky and ground background
- Styled windows with cross details
- Custom-shaped roof using clip-path
- Decorative bushes and detailed door design

## Technologies Used
- HTML5
- CSS3
- CSS Gradients
- CSS Animations (keyframes)
- CSS clip-path
- Absolute Positioning
- Google Fonts (Playfair Display)

## Usage
1. Clone the repository:
   git clone https://github.com/Vanitsas/FreeCodeCamp-All.git

2. Navigate to the project folder:
   cd "FreeCodeCamp-All/Responsive Web Design Certification/Labs/build-a-house-painting"

3. Open index.html in your browser.

## Notes
- The entire scene is built without images.
- Animations are created using CSS keyframes.
- The project focuses on positioning, layering (z-index), and visual composition.
