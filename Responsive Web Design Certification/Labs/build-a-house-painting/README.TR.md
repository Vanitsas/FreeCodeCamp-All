# Ev Boyama İnşaatı

## Proje Hakkında
Bu proje, yalnızca HTML ve CSS kullanılarak oluşturulmuş bir ev sahnesi tasarımıdır. Çatı, baca (animasyonlu duman ile), pencereler, kapı, çalılar, gökyüzü gradient arka planı ve hareketli bulutlar tamamen CSS ile hazırlanmıştır.

Bu çalışmanın amacı CSS konumlandırma, animasyon, katmanlama (z-index) ve görsel kompozisyon becerilerini geliştirmektir.

## Özellikler
- Saf HTML & CSS çizim (görsel kullanılmadı)
- Animasyonlu baca dumanı
- CSS keyframes ile hareket eden bulutlar
- Gradient gökyüzü ve zemin tasarımı
- Çapraz detaylı pencereler
- clip-path ile oluşturulmuş özel çatı şekli
- Detaylı kapı ve çalı tasarımı

## Kullanılan Teknolojiler
- HTML5
- CSS3
- CSS Gradient
- CSS Animations (keyframes)
- CSS clip-path
- Absolute Positioning
- Google Fonts (Playfair Display)
