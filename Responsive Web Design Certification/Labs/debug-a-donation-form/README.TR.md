# Bağış Formunda Hata Ayıklama

Bu proje, freeCodeCamp'ın Responsive Web Design müfredatının bir parçasıdır.

Bu lab çalışmasının amacı, bir HTML formundaki hataları ayıklamak ve form yapısını iyileştirmektir. Form; isim, e-posta, bağış miktarı ve bülten aboneliği gibi bilgileri toplar.

## Öğrenilen Konular
- HTML form hatalarını ayıklama
- Form input tiplerini doğru kullanma
- Erişilebilir form etiketleri oluşturma
- Zorunlu alanları tanımlama
- Basit ve işlevsel formlar oluşturma

## Kullanılan Teknolojiler
- HTML5
