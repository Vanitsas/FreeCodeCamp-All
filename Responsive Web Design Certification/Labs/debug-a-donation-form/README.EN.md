# Debug a Donation Form

This project is part of freeCodeCamp's Responsive Web Design curriculum.

The purpose of this lab is to practice debugging and improving an HTML form. The form collects user information such as name, email, donation amount, and newsletter subscription.

## What You Learn
- Debugging HTML form issues
- Using form input types correctly
- Creating accessible form labels
- Handling required form fields
- Building simple and functional forms

## Technologies Used
- HTML5
