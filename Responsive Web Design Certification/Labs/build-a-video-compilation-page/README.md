# Build a Video Compilation Page

This project is a simple HTML lab exercise called **Build a Video Compilation Page**.

## Description

The goal of this project is to practice semantic HTML structure by creating a video compilation page.  
The page includes a main container, headings, descriptive text, and multiple sections with embedded videos using iframes.

## Requirements Covered

- Proper use of semantic HTML elements (`main`, `section`, `h1`, `h2`, `p`)
- A single `main` element as the only child of `body`
- Multiple sections inside `main`
- Embedded videos using `iframe`
- Each iframe includes `src`, `title`, `width`, and `height` attributes

## Technologies Used

- HTML5

## Purpose

This project is part of a **lab / practice exercise** to strengthen basic HTML and page structure skills.
