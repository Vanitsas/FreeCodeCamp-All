# Design a Pricing Plans Layout Page

This project is a lab exercise for creating a **Pricing Plans Layout Page** using HTML and CSS. The page includes three pricing cards: Basic, Pro, and Premium. Each card displays a plan name and price. Flexbox is used to layout the cards responsively.

## Features

- Responsive pricing cards using Flexbox
- Three different plan types: Basic, Pro, Premium
- Flexible layout with `flex-grow` and `order` properties
