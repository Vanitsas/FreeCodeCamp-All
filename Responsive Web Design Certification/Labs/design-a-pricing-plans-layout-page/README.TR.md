# Fiyatlandırma Planları Düzeni Sayfası Tasarımı

Bu proje, HTML ve CSS kullanarak bir **Fiyatlandırma Planları Düzeni Sayfası** oluşturma laboratuvar çalışmasıdır. Sayfa, Basic, Pro ve Premium olmak üzere üç fiyat kartı içerir. Her kart, plan adını ve fiyatını gösterir. Kartların düzeni için Flexbox kullanılmıştır.

## Özellikler

- Flexbox kullanılarak responsive fiyat kartları
- Üç farklı plan türü: Basic, Pro, Premium
- `flex-grow` ve `order` özellikleri ile esnek düzen
