# Gazete Düzeni Tasarlama

Bu lab çalışmasının amacı CSS Grid kullanarak gazete tarzı bir sayfa düzeni oluşturmaktır. Birden fazla makale ve görsel klasik gazete düzenine benzer şekilde organize edilir.

## Genel Bakış

Sayfa şu bölümlerden oluşur:

- Ana gazete başlığı
- Ana haber (feature article)
- Kapak görseli
- İkincil haber
- Üç küçük haber

Bu bölümler CSS Grid Template Areas kullanılarak yerleştirilmiştir.

## Kullanılan Teknolojiler

- HTML5
- CSS3
- CSS Grid

## Layout Yapısı

Sayfa 3 sütunlu bir grid sistemi kullanır.

Bölümler:

- Üstte tam genişlikte başlık
- Ana haber ve kapak görseli
- İkincil haber
- Alt kısımda üç küçük haber

Kullanılan grid alanları:

- title
- feature-article
- cover-image
- secondary-article
- small-article1
- small-article2
- small-article3

## Bu Çalışmada Pratik Yapılan Konular

- CSS Grid ile sayfa düzeni oluşturma
- grid-template-areas kullanımı
- Çok bölümlü sayfa yapısı kurma
- Görselleri responsive kullanma
- Makaleleri düzenli bir layout içinde yerleştirme
