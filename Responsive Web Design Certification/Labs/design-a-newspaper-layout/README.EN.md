# Design a Newspaper Layout

This lab focuses on creating a newspaper-style page layout using CSS Grid. The goal is to organize multiple articles and images into a structured layout similar to a traditional newspaper.

## Overview

The page includes:

- A main newspaper title
- A featured article
- A cover image
- A secondary article
- Three smaller articles

These elements are arranged using CSS Grid Template Areas which makes the layout easier to control and visually structured.

## Technologies Used

- HTML5
- CSS3
- CSS Grid

## Layout Structure

The layout uses a 3-column grid with multiple rows.

Sections include:

- Title section across the entire top
- Featured article with a cover image
- Secondary article
- Three small articles at the bottom

Grid areas used:

- title
- feature-article
- cover-image
- secondary-article
- small-article1
- small-article2
- small-article3

## What I Practiced

- Creating layouts using CSS Grid
- Using grid-template-areas
- Structuring multi-section pages
- Organizing articles visually
- Responsive image behavior using max-width
