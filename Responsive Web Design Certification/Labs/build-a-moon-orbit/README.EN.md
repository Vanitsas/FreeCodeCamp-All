# Build a Moon Orbit

This project creates a simple animation of the Moon orbiting around the Earth using HTML and CSS.

## Project Structure

Responsive Web Design Certification/Labs/build-a-moon-orbit

## Features

- Earth positioned at the center
- Moon orbiting around the Earth
- Smooth CSS animation using `@keyframes`
- Built only with HTML and CSS

## Technologies Used

- HTML5
- CSS3
- CSS Animations

## How It Works

The Earth is positioned at the center using absolute positioning and transform.

The Moon is placed inside an orbit container that rotates continuously using a CSS animation.

The animation is defined with:

`@keyframes orbit`

This rotation creates the effect of the Moon orbiting the Earth.

## Purpose

This lab is part of the FreeCodeCamp Responsive Web Design Certification and focuses on practicing CSS positioning and animations.
