# Ay Yörüngesi Animasyonu

Bu projede HTML ve CSS kullanarak Ay'ın Dünya etrafında dönmesini gösteren basit bir animasyon oluşturulmuştur.

## Projenin Repo İçindeki Yeri

Responsive Web Design Certification/Labs/build-a-moon-orbit

## Özellikler

- Dünya merkeze yerleştirilmiştir
- Ay Dünya'nın etrafında döner
- `@keyframes` ile akıcı CSS animasyonu
- Sadece HTML ve CSS kullanılmıştır

## Kullanılan Teknolojiler

- HTML5
- CSS3
- CSS Animations

## Nasıl Çalışır

Dünya absolute positioning ve transform kullanılarak kapsayıcının ortasına yerleştirilir.

Ay ise orbit adlı kapsayıcının içinde bulunur ve bu kapsayıcı CSS animasyonu ile sürekli döndürülür.

Animasyon şu şekilde tanımlanır:

`@keyframes orbit`

Bu dönüş Ay'ın Dünya etrafında döndüğü izlenimini oluşturur.
