# Ödeme Sayfası Oluşturun

Bu workshop, freeCodeCamp'ın Responsive Web Design müfredatının bir parçasıdır.

Bu çalışmada, bir e-ticaret senaryosu için işlevsel bir ödeme sayfası oluşturacaksınız. Doğru form etiketleme, anlamsal HTML ve ARIA özelliklerini kullanarak erişilebilirlik uygulayacaksınız.

## Öğrenilen Konular
- Anlamsal HTML ile checkout sayfası oluşturma
- Etiketler ve ARIA ile erişilebilir formlar oluşturma
- E-ticaret akışı için içerik düzenleme

## Kullanılan Teknolojiler
- HTML5
- ARIA özellikleri

## Kaynaklar
- freeCodeCamp: Build a Checkout Page
