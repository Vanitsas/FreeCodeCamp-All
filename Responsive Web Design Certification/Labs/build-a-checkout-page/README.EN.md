# Build a Checkout Page

This lab is part of freeCodeCamp's Responsive Web Design curriculum.

In this lab, you will create a functional checkout page for an e-commerce scenario. You will practice proper form labeling, semantic HTML, and accessibility using ARIA attributes.

## Objectives
- Structure a checkout page with semantic HTML
- Create accessible forms with labels and ARIA
- Organize content for e-commerce flow

## Technologies Used
- HTML5
- ARIA attributes
