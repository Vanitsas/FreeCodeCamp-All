# Build a Tribute Page

## Project Overview
This project is a tribute page dedicated to **Ada Lovelace**, widely recognized as the world's first computer programmer. The page showcases her life milestones, achievements, and contributions to computing. The goal of this project is to practice **HTML structure**, **CSS styling**, and **responsive layout design** while creating a visually appealing tribute page.

## Features
- Responsive design for desktop and mobile devices
- Header with decorative ornament and main title
- Portrait image with caption
- Timeline section displaying key milestones in Ada Lovelace's life
- Pull quote section highlighting notable words
- Footer with a link to her Wikipedia page
- Custom fonts and color palette for a visually distinct style

## Technologies Used
- **HTML5** – for semantic structure
- **CSS3** – for styling and layout
- **Google Fonts** – 'Playfair Display', 'IM Fell English', and 'Cormorant Garamond'
