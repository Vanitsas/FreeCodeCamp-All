# Build an Event Hub

This project is part of a freeCodeCamp HTML Lab.

The purpose of this project is to practice semantic HTML by building an event web page. The focus is on using elements like `header`, `main`, `footer`, `section`, `article`, and `img`.

## What You Learn
- HTML5 page structure
- Semantic HTML elements (`header`, `main`, `footer`, `section`, `article`)
- Headings and content hierarchy (`h1`, `h2`, `h3`)
- Navigation and links (`nav`, `a`)
- Images and alternative text (`img`, `alt`)

## Technologies Used
- HTML5
