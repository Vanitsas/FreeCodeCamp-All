# Bir Etkinlik Merkezi Oluşturun

Bu proje, freeCodeCamp HTML Lab çalışmasının bir parçası olarak hazırlanmıştır.

Bu çalışmanın amacı, semantik HTML kullanarak bir etkinlik web sayfası oluşturmaktır. Özellikle `header`, `main`, `footer`, `section`, `article` ve `img` öğeleri üzerinde durulmuştur.

## Öğrenilen Konular
- HTML5 sayfa yapısı
- Semantik HTML öğeleri (`header`, `main`, `footer`, `section`, `article`)
- Başlık ve içerik hiyerarşisi (`h1`, `h2`, `h3`)
- Navigasyon ve linkler (`nav`, `a`)
- Görseller ve alternatif metin (`img`, `alt`)

## Kullanılan Teknolojiler
- HTML5
