# Build an HTML Audio and Video Player

This is a simple HTML project that demonstrates how to use the `<audio>` and `<video>` elements in a web page. It was completed as part of the **FreeCodeCamp** curriculum.

## Features

- Plays a video using the HTML `<video>` element
- Plays audio using the HTML `<audio>` element
- Video element includes:
  - Controls
  - Width set to 640
  - Source file with type specified
- Audio element includes:
  - Controls
  - Loop enabled
  - Source file specified
- Fallback text for unsupported browsers

## Demo

- Video source: [Map Method Video](https://cdn.freecodecamp.org/curriculum/labs/what-is-the-map-method-and-how-does-it-work.mp4)  
- Audio source: [Sailing Away](https://cdn.freecodecamp.org/curriculum/js-music-player/sailing-away.mp3)  

## How to Use

1. Clone this repository or download the files.
2. Open `index.html` in your browser.
3. You can play the video and audio directly in the browser.
