# Build a Book Catalog Table

This project is part of a freeCodeCamp HTML Lab.

The purpose of this project is to practice creating semantic and well-structured HTML tables. The focus is on organizing tabular data clearly using table sections such as `thead`, `tbody`, and `tfoot`.

## What You Learn
- HTML table structure
- Using `table`, `thead`, `tbody`, and `tfoot`
- Table rows and cells (`tr`, `th`, `td`)
- Using `colspan` for summary rows

## Technologies Used
- HTML5
