# Kitap Kataloğu Masası Oluşturun

Bu proje, freeCodeCamp HTML Lab çalışmasının bir parçası olarak hazırlanmıştır.

Bu çalışmanın amacı, semantik ve düzenli HTML tabloları oluşturmaktır. Tablo verilerinin `thead`, `tbody` ve `tfoot` bölümleri kullanılarak doğru şekilde yapılandırılması hedeflenmiştir.

## Öğrenilen Konular
- HTML tablo yapısı
- `table`, `thead`, `tbody` ve `tfoot` kullanımı
- Tablo satırları ve hücreleri (`tr`, `th`, `td`)
- Özet satırları için `colspan` kullanımı

## Kullanılan Teknolojiler
- HTML5
