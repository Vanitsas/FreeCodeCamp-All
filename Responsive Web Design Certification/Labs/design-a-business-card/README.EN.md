# Design a Business Card
This is a simple **Business Card** project created as part of the FreeCodeCamp curriculum. The project demonstrates basic HTML and CSS skills by creating a personal card with contact information and social media links.

### Features
- Profile image
- Full name, designation, and company
- Contact information (email and phone)
- Portfolio link
- Social media links (Twitter, LinkedIn, GitHub)

### Technologies Used
- HTML
- CSS

### How to Run
1. Clone this repository or download the files.
2. Open the `index.html` file in your web browser.
