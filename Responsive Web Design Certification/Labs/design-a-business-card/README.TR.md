# Kartvizit Tasarımı
Bu proje, FreeCodeCamp müfredatının bir parçası olarak oluşturulmuş basit bir **Kartvizit** çalışmasıdır. HTML ve CSS temel bilgilerini kullanarak kişisel bir kart oluşturmayı ve iletişim ile sosyal medya bağlantılarını göstermeyi amaçlar.

### Özellikler
- Profil fotoğrafı
- İsim, unvan ve şirket bilgisi
- İletişim bilgileri (e-posta ve telefon)
- Portföy linki
- Sosyal medya bağlantıları (Twitter, LinkedIn, GitHub)

### Kullanılan Teknolojiler
- HTML
- CSS

### Çalıştırma
1. Bu depoyu klonlayın veya dosyaları indirin.
2. `index.html` dosyasını web tarayıcınızda açın.
