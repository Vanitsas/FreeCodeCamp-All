# Build-a-newspaper-article

This project demonstrates a **newspaper-style article layout** using HTML and CSS. The layout mimics a traditional newspaper with headlines, sub-headlines, author info, and article text with a styled first letter.

## Features

- Newspaper title and date
- Headline and sub-headline
- Author information
- Article paragraphs with justified text
- Styled first letter for a traditional newspaper look
- Responsive design using CSS for spacing and typography

## HTML Structure

- `div.newspaper` – main container
- `div.name` – newspaper title
- `div.date` – date and volume info
- `div.headline` – main headline
- `div.sub-headline` – subtitle of the article
- `div.author` – author information
- `p.text` – paragraphs of the article

## CSS Highlights

- Custom font via Google Fonts: *Open Sans*
- Box shadow for newspaper container
- Borders for headings and newspaper name
- `text-indent` and `::first-letter` pseudo-element for classic newspaper effect
- Flexbox and padding for responsive layout

## Usage

1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. Observe the newspaper layout and styling.
