# Film İnceleme Sayfası Tasarımı

Bu proje, freeCodeCamp HTML Laboratuvarı kapsamında oluşturulmuştur.

Projenin amacı, semantik ve iyi yapılandırılmış bir film inceleme sayfası oluşturma pratiği yapmaktır. Odak, `main`, `h1`, `h2`, `p`, `ul` ve `li` gibi semantik HTML öğelerini kullanarak içeriği düzenli ve erişilebilir şekilde sunmaktır.

## Öğrenecekleriniz
- Semantik HTML yapısı
- Başlıklar, paragraflar, listeler ve resimleri kullanma
- Resimler için açıklayıcı alt metin ekleme
- Erişilebilir film puanlaması gösterimi

## Kullanılan Teknolojiler
- HTML5
