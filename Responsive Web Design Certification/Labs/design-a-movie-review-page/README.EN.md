# Design a Movie Review Page

This project is part of a freeCodeCamp HTML Lab.

The purpose of this project is to practice creating a semantic and well-structured movie review page. The focus is on organizing content clearly using semantic HTML elements such as `main`, `h1`, `h2`, `p`, `ul`, and `li`.

## What You Learn
- Semantic HTML structure
- Using headings, paragraphs, lists, and images
- Adding descriptive alt text for images
- Displaying accessible movie ratings

## Technologies Used
- HTML5
