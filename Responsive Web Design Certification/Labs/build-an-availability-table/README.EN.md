# Build an Availability Table

This lab project is part of the FreeCodeCamp Responsive Web Design learning path. The goal is to build a clean availability schedule using HTML tables and CSS styling.

The interface displays a weekly availability schedule where each cell represents a specific time slot for a day. Different color levels indicate availability from 0 (not available) to 5 (fully available).

## Features

- Semantic HTML table structure
- Weekly schedule layout
- Color-based availability indicators
- CSS variables for consistent color management
- Responsive table container
- Visual legend explaining availability levels
- Modern typography with Google Fonts

## Technologies Used

- HTML5
- CSS3
- CSS Variables
- Flexbox
- Responsive Design
- Google Fonts
