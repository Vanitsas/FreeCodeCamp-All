# Kullanılabilirlik Tablosu Oluşturma

Bu lab projesi FreeCodeCamp Responsive Web Design öğrenme yolunun bir parçasıdır. Amaç HTML tabloları ve CSS kullanarak haftalık bir kullanılabilirlik tablosu oluşturmaktır.

Arayüz haftalık bir programı gösterir. Her hücre belirli bir gün ve saat için kullanılabilirlik durumunu temsil eder. Farklı renk seviyeleri 0 (müsait değil) ile 5 (tam müsait) arasındaki durumları gösterir.

## Özellikler

- Anlamlı (semantic) HTML tablo yapısı
- Haftalık program düzeni
- Renk tabanlı kullanılabilirlik göstergeleri
- Tutarlı tasarım için CSS değişkenleri
- Responsive tablo kapsayıcısı
- Kullanılabilirlik seviyelerini açıklayan legend
- Google Fonts ile modern tipografi

## Kullanılan Teknolojiler

- HTML5
- CSS3
- CSS Variables
- Flexbox
- Responsive Tasarım
- Google Fonts

## Proje Yapısı

build-an-availability-table

## Bu Projede Pratik Edilenler

- `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>` kullanımı
- CSS ile tablo tasarımı geliştirme
- CSS değişkenleri kullanarak tasarım sistemi oluşturma
- Renklerle veri görselleştirme
- Geniş tablolar için responsive yapı oluşturma
- Arayüzde legend kullanımı
