# build-a-personal-portfolio

## Proje Hakkında
Bu proje, projelerinizi sergileyen, bir karşılama bölümü sunan ve iletişim bilgilerini sağlayan kişisel bir portföy web sitesidir. Site responsive tasarıma sahiptir ve bölümlere kolay erişim için sabit bir gezinme çubuğu içerir.

## Proje Yapısı
build-a-personal-portfolio/
├── index.html
├── styles.css

## Özellikler
- **Gezinme Çubuğu:** Üstte sabit, Home (Ana Sayfa), Projects (Projeler) ve Contact (İletişim) bölümlerine bağlantılar içerir.
- **Karşılama Bölümü:** Kendinizi başlık ve kısa bir açıklama ile tanıtmanızı sağlar.
- **Projeler Bölümü:** Projelerinizi tıklanabilir bağlantılarla responsive bir grid içinde gösterir.
- **İletişim Bölümü:** GitHub profilinize bağlantı sağlar.
- **Responsive Tasarım:** Masaüstü ve mobil cihazlarda uyumlu çalışır.
