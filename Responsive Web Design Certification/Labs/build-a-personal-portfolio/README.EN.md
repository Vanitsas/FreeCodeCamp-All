# build-a-personal-portfolio

## Project Overview
This project is a personal portfolio website that showcases your projects, provides a welcome section, and offers contact information. It is responsive and includes a fixed navigation bar for easy access to sections.

## Project Structure
build-a-personal-portfolio/
├── index.html
├── styles.css

## Features
- **Navigation Bar:** Fixed at the top with links to Home, Projects, and Contact sections.
- **Welcome Section:** Introduces yourself with a title and a short description.
- **Projects Section:** Displays your projects in a responsive grid with clickable links.
- **Contact Section:** Provides a link to your GitHub profile.
- **Responsive Design:** Works on desktops and mobile devices with media queries.
