# Build a Multimedia Player

This project is part of a freeCodeCamp HTML Lab.

The purpose of this project is to practice embedding multimedia content in a webpage using semantic HTML. The focus is on creating accessible audio and video players with proper labels, subtitles, and structure.

## What You Learn
- Embedding audio and video using `<audio>` and `<video>` tags
- Using `source` and `track` elements for media files and subtitles
- Providing accessible labels with `aria-label`
- Organizing content using semantic `<section>` elements
- Creating transcripts for video content

## Technologies Used
- HTML5
