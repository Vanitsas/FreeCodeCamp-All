# Build a JavaScript Trivia Bot

A beginner JavaScript lab completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this lab, a trivia bot is built that shares fun facts about JavaScript. The bot introduces itself, delivers coding facts one by one, and says goodbye at the end.

## 🧠 Concepts Practiced

- Declaring variables with `let`
- Reassigning `let` variables
- Template literals (`` ` `` backtick strings with `${}`)
- Using `console.log()` to output messages

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```

## 📄 Expected Output

```
Hello! I'm your coding fun fact guide!
My name is Codey and I live on Internet.
My favorite programming language is JavaScript.
JavaScript was created in just 10 days!
JavaScript can be used for both frontend and backend development.
JavaScript is one of the most popular programming languages in the world.
It was fun sharing these facts with you. Goodbye! - Codey from Internet.
```
