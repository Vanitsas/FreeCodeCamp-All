# E-posta Maskeleyici Oluştur

[freeCodeCamp](https://www.freecodecamp.org/) müfredatı kapsamında tamamlanmış başlangıç seviyesi bir JavaScript laboratuvar çalışmasıdır.

## 📌 Hakkında

Bu laboratuvarda bir e-posta adresinin kullanıcı adı kısmını gizlilik amacıyla maskeleyen bir fonksiyon oluşturulmuştur. Kullanıcı adının ilk ve son karakterleri görünür tutulurken ortadaki karakterler yıldız işareti (`*`) ile değiştirilir. 2 karakter veya daha kısa kullanıcı adları özel durum olarak ele alınmıştır.

## 🧠 Çalışılan Kavramlar

- Fonksiyon tanımlama ve çağırma
- Bir karakterin konumunu bulmak için `indexOf()` kullanımı
- String'in bölümlerini ayıklamak için `slice()` kullanımı
- Yıldız dizisi oluşturmak için `repeat()` kullanımı
- İndeksle karaktere erişmek için köşeli parantez notasyonu (`string[index]`)
- Kenar durum yönetimi için `if` koşulu
- String birleştirme

## 💻 Nasıl Çalıştırılır

1. Kodu tarayıcının geliştirici konsoluna yapıştır, veya
2. Node.js ile çalıştır:

```bash
node index.js
```
