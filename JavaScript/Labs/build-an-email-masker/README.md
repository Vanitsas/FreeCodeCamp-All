# Build an Email Masker

A beginner JavaScript lab completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this lab, a function is built that masks the username part of an email address for privacy. The first and last characters of the username are kept visible while the middle characters are replaced with asterisks (`*`). Short usernames of 2 characters or less are handled as a special case.

## 🧠 Concepts Practiced

- Declaring and calling functions
- `indexOf()` to find the position of a character
- `slice()` to extract parts of a string
- `repeat()` to generate a string of asterisks
- Bracket notation to access characters by index (`string[index]`)
- `if` conditional for edge case handling
- String concatenation

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```

## 📄 Example Output

```
a*******e@example.com
```
