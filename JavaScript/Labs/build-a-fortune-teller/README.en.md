# Build a Fortune Teller

A beginner JavaScript lab completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this lab, a fortune teller app is built that randomly selects and displays one of five fortunes. A random number is generated to pick a fortune using a chain of `if...else if` statements.

## 🧠 Concepts Practiced

- `let` variable declarations
- `Math.floor()` and `Math.random()` to generate a random integer in a range
- `if...else if` conditional chains
- Selecting a value based on a random number

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```

## 📄 Expected Output

One of the following is printed at random:

```
Your cat will look very cuddly today.
The weather will be nice tomorrow.
Be cautious of your new neighbors.
You will find a new hobby soon.
It would be wise to avoid the color red today.
```
