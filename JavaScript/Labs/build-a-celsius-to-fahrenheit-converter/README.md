# Build a Celsius to Fahrenheit Converter

A beginner JavaScript lab completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this lab, a function is built that converts a temperature value from Celsius to Fahrenheit using the standard conversion formula.

**Formula:**
```
fahrenheit = celsius * (9/5) + 32
```

## 🧠 Concepts Practiced

- Declaring and calling functions
- Applying a mathematical formula in code
- Returning a computed numeric value from a function
- Arithmetic operators (`*`, `/`, `+`)

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```
