# Celsius'tan Fahrenheit'a Dönüştürücü Oluştur

[freeCodeCamp](https://www.freecodecamp.org/) müfredatı kapsamında tamamlanmış başlangıç seviyesi bir JavaScript laboratuvar çalışmasıdır.

## 📌 Hakkında

Bu laboratuvarda standart dönüşüm formülü kullanılarak bir sıcaklık değerini Celsius'tan Fahrenheit'a çeviren bir fonksiyon oluşturulmuştur.

**Formül:**
```
fahrenheit = celsius * (9/5) + 32
```

## 🧠 Çalışılan Kavramlar

- Fonksiyon tanımlama ve çağırma
- Matematiksel bir formülü koda uygulama
- Fonksiyondan hesaplanmış sayısal değer döndürme
- Aritmetik operatörler (`*`, `/`, `+`)

## 💻 Nasıl Çalıştırılır

1. Kodu tarayıcının geliştirici konsoluna yapıştır, veya
2. Node.js ile çalıştır:

```bash
node index.js
```
