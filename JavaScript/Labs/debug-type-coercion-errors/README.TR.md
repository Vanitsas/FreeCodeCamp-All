# Hatalı Uygulamadaki Tip Dönüşümü Hatalarını Düzelt

[freeCodeCamp](https://www.freecodecamp.org/) müfredatı kapsamında tamamlanmış başlangıç seviyesi bir JavaScript laboratuvar çalışmasıdır.

## 📌 Hakkında

Bu laboratuvarda bozuk bir uygulamadaki tip dönüşümü (type coercion) hataları tespit edilip düzeltilmiştir. Orijinal kodda sayı beklenen yerlerde string değerler kullanılmıştı; bu durum JavaScript'in otomatik tip dönüşümü nedeniyle yanlış sonuçlar üretiyordu. Tüm değişkenler düzgün sayı tipine çevrildi.

## 🧠 Çalışılan Kavramlar

- JavaScript tip dönüşümünü (type coercion) anlamak
- Aritmetik işlemlerde sayı ve string değerlerinin farkı
- Tip kaynaklı hataları hata ayıklama ve düzeltme
- `const` ile değişken tanımlama
- `${}` ile template literal kullanımı

## 💻 Nasıl Çalıştırılır

1. Kodu tarayıcının geliştirici konsoluna yapıştır, veya
2. Node.js ile çalıştır:

```bash
node index.js
```

## 📄 Beklenen Çıktı

```
5 + 10 = 15
8 - 5 = 3
1 + 5 = 6
0 + 8 = 8
10 * 2 = 20
0 + 22 = 22
```
