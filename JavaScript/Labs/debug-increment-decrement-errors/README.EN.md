# Debug Increment and Decrement Operator Errors in a Buggy App

A beginner JavaScript lab completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this lab, increment and decrement operator bugs in a broken app are identified and fixed. The key concept is understanding the difference between pre-increment (`++x`) and post-increment (`x++`) operators, and how each affects the assigned value.

## 🧠 Concepts Practiced

- Pre-increment (`++x`) — increments first, then assigns
- Post-increment (`x++`) — assigns first, then increments
- Pre-decrement (`--x`) — decrements first, then assigns
- Post-decrement (`x--`) — assigns first, then decrements
- Debugging operator-related logic errors
- Template literals with `${}`

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```
