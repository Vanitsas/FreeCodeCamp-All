# Hatalı Uygulamadaki Artırma ve Azaltma Operatörü Hatalarını Düzelt

[freeCodeCamp](https://www.freecodecamp.org/) müfredatı kapsamında tamamlanmış başlangıç seviyesi bir JavaScript laboratuvar çalışmasıdır.

## 📌 Hakkında

Bu laboratuvarda bozuk bir uygulamadaki artırma ve azaltma operatörü hataları tespit edilip düzeltilmiştir. Temel konu, ön-artırma (`++x`) ile son-artırma (`x++`) operatörleri arasındaki farkı ve her birinin atanan değeri nasıl etkilediğini anlamaktır.

## 🧠 Çalışılan Kavramlar

- Ön-artırma (`++x`) — önce artırır, sonra atar
- Son-artırma (`x++`) — önce atar, sonra artırır
- Ön-azaltma (`--x`) — önce azaltır, sonra atar
- Son-azaltma (`x--`) — önce atar, sonra azaltır
- Operatör kaynaklı mantık hatalarını hata ayıklama
- `${}` ile template literal kullanımı

## 💻 Nasıl Çalıştırılır

1. Kodu tarayıcının geliştirici konsoluna yapıştır, veya
2. Node.js ile çalıştır:

```bash
node index.js
```
