# Build a Sentence Maker

A beginner JavaScript lab completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this lab, a dynamic sentence maker is built using variables and template literals. The same story template is reused with different words to generate two unique stories.

## 🧠 Concepts Practiced

- Declaring variables with `let`
- Reassigning `let` variables
- Template literals (`` ` `` backtick strings with `${}`)
- Reusing a sentence structure with different values
- Using `console.log()` to output messages

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```

## 📄 Expected Output

```
First story: Once upon a time, there was a(n) brave dragon who loved to eat pizza. The dragon lived in a mountain and had fiery nostrils that blew fire when it was angry.
Second story: Once upon a time, there was a(n) funny wizard who loved to eat cookies. The wizard lived in a forest and had sparkly nostrils that blew fire when it was dancing.
```
