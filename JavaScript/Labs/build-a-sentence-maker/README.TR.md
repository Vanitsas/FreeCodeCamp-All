# Cümle Yapıcı Oluştur

[freeCodeCamp](https://www.freecodecamp.org/) müfredatı kapsamında tamamlanmış başlangıç seviyesi bir JavaScript laboratuvar çalışmasıdır.

## 📌 Hakkında

Bu laboratuvarda değişkenler ve template literal'lar kullanılarak dinamik bir cümle yapıcı oluşturulmuştur. Aynı hikaye şablonu farklı kelimelerle yeniden kullanılarak iki benzersiz hikaye üretilmiştir.

## 🧠 Çalışılan Kavramlar

- `let` ile değişken tanımlama
- `let` değişkenlerine yeniden değer atama
- Template literal kullanımı (`` ` `` backtick ve `${}` sözdizimi)
- Aynı cümle yapısını farklı değerlerle yeniden kullanma
- `console.log()` ile ekrana çıktı verme

## 💻 Nasıl Çalıştırılır

1. Kodu tarayıcının geliştirici konsoluna yapıştır, veya
2. Node.js ile çalıştır:

```bash
node index.js
```

## 📄 Beklenen Çıktı

```
First story: Once upon a time, there was a(n) brave dragon who loved to eat pizza. The dragon lived in a mountain and had fiery nostrils that blew fire when it was angry.
Second story: Once upon a time, there was a(n) funny wizard who loved to eat cookies. The wizard lived in a forest and had sparkly nostrils that blew fire when it was dancing.
```
