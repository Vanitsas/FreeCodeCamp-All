# Build a Calculator

A beginner JavaScript workshop completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this workshop, a calculator is built using JavaScript functions. Each arithmetic operation is written as a separate function, covering addition, subtraction, multiplication, division, exponentiation, and square roots.

## 🧠 Concepts Practiced

- Declaring and calling functions
- Returning values from functions
- Arithmetic operators (`+`, `-`, `*`, `/`, `**`)
- Ternary operator for handling division by zero
- `Math.sqrt()` for square root calculations
- Exponentiation operator (`**`)

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```
