# Hesap Makinesi Oluştur

[freeCodeCamp](https://www.freecodecamp.org/) müfredatı kapsamında tamamlanmış başlangıç seviyesi bir JavaScript atölyesidir.

## 📌 Hakkında

Bu atölyede JavaScript fonksiyonları kullanılarak bir hesap makinesi oluşturulmuştur. Her aritmetik işlem ayrı bir fonksiyon olarak yazılmış; toplama, çıkarma, çarpma, bölme, üs alma ve karekök hesaplama işlemleri ele alınmıştır.

## 🧠 Çalışılan Kavramlar

- Fonksiyon tanımlama ve çağırma
- Fonksiyonlardan değer döndürme (`return`)
- Aritmetik operatörler (`+`, `-`, `*`, `/`, `**`)
- Sıfıra bölme durumunu ele almak için ternary operatör
- Karekök hesaplama için `Math.sqrt()` kullanımı
- Üs alma operatörü (`**`)

## 💻 Nasıl Çalıştırılır

1. Kodu tarayıcının geliştirici konsoluna yapıştır, veya
2. Node.js ile çalıştır:

```bash
node index.js
```
