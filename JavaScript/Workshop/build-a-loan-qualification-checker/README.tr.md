# Kredi Uygunluk Denetleyicisi Oluştur

[freeCodeCamp](https://www.freecodecamp.org/) müfredatı kapsamında tamamlanmış başlangıç seviyesi bir JavaScript atölyesidir.

## 📌 Hakkında

Bu atölyede, bir kişinin yıllık geliri ve kredi skoruna göre hangi kredilere uygun olduğunu belirleyen bir kredi uygunluk denetleyicisi oluşturulmuştur. Fonksiyon; dubleks, kondominiyum ve araç kredisi olmak üzere üç kredi türü için minimum eşik değerlere göre kontrol yapar.

## 🧠 Çalışılan Kavramlar

- Parametreli fonksiyon tanımlama
- `if...else if...else` koşul zinciri
- Birden fazla koşulu kontrol etmek için mantıksal VE operatörü (`&&`)
- Değerleri sabitlerle karşılaştırma
- Fonksiyonlardan string döndürme

## 💻 Nasıl Çalıştırılır

1. Kodu tarayıcının geliştirici konsoluna yapıştır, veya
2. Node.js ile çalıştır:

```bash
node index.js
```
