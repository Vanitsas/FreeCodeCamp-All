# Build a Loan Qualification Checker

A beginner JavaScript workshop completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this workshop, a loan qualification checker is built that determines which loans a person qualifies for based on their annual income and credit score. The function checks against minimum thresholds for three loan types: duplex, condo, and car loans.

## 🧠 Concepts Practiced

- Declaring functions with parameters
- `if...else if...else` conditional chains
- Logical AND operator (`&&`) to check multiple conditions
- Comparing values against constants
- Returning strings from functions

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```

## 📄 Expected Output

```
You qualify for a duplex, condo, and car loan.
You qualify for a condo and car loan.
You qualify for a car loan.
You don't qualify for any loans.
```
