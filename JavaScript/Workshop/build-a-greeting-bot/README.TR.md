# Selamlama Botu Oluştur

[freeCodeCamp](https://www.freecodecamp.org/) müfredatı kapsamında tamamlanmış başlangıç seviyesi bir JavaScript atölyesidir.

## 📌 Hakkında

Bu atölyede temel JavaScript kavramları kullanılarak basit bir selamlama botu oluşturulmuştur. Bot kendini tanıtır, nerede yaşadığını söyler ve program boyunca takma adını günceller.

## 🧠 Çalışılan Kavramlar

- `let` ve `const` ile değişken tanımlama
- `let` değişkenlerine yeniden değer atama
- String birleştirme (concatenation)
- `console.log()` ile ekrana çıktı verme

## 💻 Nasıl Çalıştırılır

1. Kodu tarayıcının geliştirici konsoluna yapıştır, veya
2. Node.js ile çalıştır:

```bash
node index.js
```
