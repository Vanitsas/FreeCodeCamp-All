# Build a String Transformer

A beginner JavaScript workshop completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this workshop, string transformation methods are explored. The program replaces words in sentences, replaces all occurrences of a word at once, and repeats a string multiple times to build a new sentence.

## 🧠 Concepts Practiced

- Using `replace()` to replace the first occurrence of a substring
- Using `replaceAll()` to replace all occurrences of a substring
- Using `repeat()` to repeat a string multiple times
- Combining `repeat()` with `trimEnd()` to clean up trailing whitespace
- Template literals with `${}`

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```

## 📄 Expected Output

```
Original string:
I love cats.
After using the replace() method:
I love dogs.
Original sentence:
I love cats and cats are so much fun!
Replacing all occurrences of cats with dogs:
I love dogs and dogs are so much fun!
Original learning sentence:
I love learning!
love love love
I love love love learning.
```
