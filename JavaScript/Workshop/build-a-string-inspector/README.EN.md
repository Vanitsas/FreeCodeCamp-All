# Build a String Inspector

A beginner JavaScript workshop completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this workshop, two essential JavaScript string methods are explored: `includes()` and `slice()`. The program checks whether certain words exist in a sentence and extracts specific parts of a string.

## 🧠 Concepts Practiced

- Using `includes()` to check if a string contains a substring
- Understanding that `includes()` is case-sensitive
- Using `slice()` to extract parts of a string by index
- Using negative index with `slice()` to get characters from the end
- Declaring variables with `const`
- Template literals with `${}`

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```
