# Mantık Denetleyici Uygulama Oluştur

[freeCodeCamp](https://www.freecodecamp.org/) müfredatı kapsamında tamamlanmış başlangıç seviyesi bir JavaScript atölyesidir.

## 📌 Hakkında

Bu atölyede boolean değerler ve koşullu ifadeler kullanılarak basit bir mantık denetleyici uygulama oluşturulmuştur. Program, Timmy adlı bir kişi hakkında çeşitli koşulları kontrol eder ve bu koşulların doğru ya da yanlış olmasına göre farklı mesajlar yazdırır.

## 🧠 Çalışılan Kavramlar

- Boolean değerler (`true` / `false`)
- `if` ifadesi
- `if...else` ifadesi
- Karşılaştırma operatörleri (`>=`)
- `const` ile değişken tanımlama

## 💻 Nasıl Çalıştırılır

1. Kodu tarayıcının geliştirici konsoluna yapıştır, veya
2. Node.js ile çalıştır:

```bash
node index.js
```
