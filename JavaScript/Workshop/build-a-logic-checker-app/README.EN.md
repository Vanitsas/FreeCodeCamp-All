# Build a Logic Checker App

A beginner JavaScript workshop completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this workshop, a simple logic checker app is built using boolean values and conditional statements. The program checks various conditions about a person named Timmy and prints different messages based on whether those conditions are true or false.

## 🧠 Concepts Practiced

- Boolean values (`true` / `false`)
- `if` statements
- `if...else` statements
- Comparison operators (`>=`)
- Using `const` for variable declarations

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```

## 📄 Expected Output

```
Timmy is employed as a developer.
Timmy is not old enough to drive.
```
