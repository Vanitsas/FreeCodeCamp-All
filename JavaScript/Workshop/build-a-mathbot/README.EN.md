# Build a Mathbot

A beginner JavaScript workshop completed as part of the [freeCodeCamp](https://www.freecodecamp.org/) curriculum.

## 📌 About

In this workshop, a math teaching bot is built to demonstrate the most commonly used JavaScript `Math` object methods. The bot walks through random number generation, rounding, and finding maximum/minimum values.

## 🧠 Concepts Practiced

- `Math.random()` — generates a pseudo-random number between 0 and 1
- Generating a random number within a custom range using `Math.random()`
- `Math.floor()` — rounds a number down to the nearest integer
- Generating a random integer within a range using `Math.floor()` + `Math.random()`
- `Math.ceil()` — rounds a number up to the nearest integer
- `Math.round()` — rounds a number to the nearest integer
- `Math.max()` — returns the largest value from a set of numbers
- `Math.min()` — returns the smallest value from a set of numbers
- Template literals with `${}`

## 💻 How to Run

1. Copy the code into your browser's developer console, or
2. Run it with Node.js:

```bash
node index.js
```
